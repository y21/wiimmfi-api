Wiimmfi API Documentation

This is an unofficial API and is not affiliated with the original online service wiimmfi. If one of the maintainers want me to change something on this project, feel free to contact me by either sending a direct message to y21#0909 (discord) or by joining the discord server.

This API is public, means you don't need a token or anything to use this service. Most of these endpoints use regular expressions to scrape information from the wiimmfi website. What that means is that if the website will be ever down or not accessable, this API won't work, too.
This service should give you accuate information, but it is not guaranteed as you may have seen in the mkw/amount endpoint.
If the frontend code will change, all regular expressions will more likely stop working, so I have to pray that they don't edit the code every few days.
